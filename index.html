ungadget by reid

Transform a OpenSocial Gadget Specification's HTML into inline HTML.

Currently, the Yahoo! Application Platform does not support Gadget XML despite supporting the OpenSocial 0.8 JavaScript API. In addition, you must include all scripts and stylesheets inlineâ€“external resources are not yet allowed. Ungadget wants to make porting an existing gadget to YAP v1.0 a bit easier. Ungadget extracts the HTML content from your gadget, downloads all external scripts and stylesheets and gives you a "flattened" HTML fragment with these resources all embedded inline.

Dependencies

Ungadget has been tested to work on PHP 5.2.6 and requires the cURL extension.
